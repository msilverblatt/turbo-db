# Semantic Image Search

Search your photos using natural language, powered by CLIP + turboquant-db.

## Setup

```bash
pip install -r requirements.txt
```

## Usage

```bash
python app.py --images /path/to/your/photos
```

Then open http://localhost:5000 and start searching.

## Example queries

- "sunset at the beach"
- "dog playing in the snow"
- "city skyline at night"
- "people laughing at a party"
- "a cup of coffee on a wooden table"
- "mountains with fog"
